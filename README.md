## Notes

### HTML
* `<p class="class1 class2">hello</p>` More than one class can be selected (space separated).

### CSS
* `element` (ex: p)
* `#id`
* `.class` (ex: .myClass)
* `element.class` (ex: p.myClass)
* grouping (`,`)
* descendant selector (`space`)
* child selector (`>`)
* adjacent sibling selector (`+`)
* general sibling selector (`~`)
